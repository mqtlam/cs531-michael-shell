I. HOW TO RUN

To run one search instance on one problem:
	./run.py type heuristic initialstate

	where for type, 1 = A* 
			2 = RBFS 

	for heuristic, 	1 = Admissible
			2 = Non-admissible
	
	and initialstate,
		use the following format: "0123"
		where the disks are on peg A and the top disk is the leftmost character

- Linux (we coded using Linux)
